# UI Component Library

This is the component library for the Systematic Review Data Extraction Tool.

## Technologies

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **shadcn/ui** - Component foundation
- **Radix UI** - Accessible primitives

## Theme

The application supports both light and dark modes. You can switch themes using the toolbar at the top.

## Getting Started

Browse the components in the sidebar to see all available UI elements.
Each component includes:
- Interactive controls to modify props
- Multiple variants and examples
- Usage documentation
- Accessibility notes

## Component Categories

### Basic Components
Form inputs and basic UI elements like buttons, inputs, checkboxes, and switches.

### Feedback Components
Components for user feedback including badges, alerts, toasts, and progress indicators.

### Layout Components
Structural components like cards, separators, and resizable panels.

### Overlay Components
Modal dialogs, popovers, tooltips, and other overlay elements.

### Navigation Components
Tabs, accordions, dropdowns, and navigation menus.

### Selection Components
Date pickers, select dropdowns, and command palettes.

### Display Components
Avatars, tables, and other display components.
